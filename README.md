#### The River

## An exploratory app to figure out what factors influence
## a users viewing pleasure the most.

## QUICK START

Assumes you have the database setup already.
Navigate to the top directory and type in:
'npm start'
then press enter.

### Tools Used:
Jest
Webpack
VueJS 2.6
MySQL Server
